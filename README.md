## Linux Commands
"folder"=the name of folder
"file"=the name of file
1. A
`pwd`: Print working folder = path of current location.
```
cd "folder"
cd ./"folder"
cd <where>/"folder"
cd ..
cd ~
cd /
pwd
```
2. B
```
ls
ls -a
ls -l
ls -al
```
3. C
```
mkdir "folder"
touch "file"
```
4. D
```
cp "file1" "file2"
cp -r "folder1" "folder2"
rsync -avh "file1" "file2"
```
